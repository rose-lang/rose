Rose is a differentiable programming for the web, inspired by [Dex][].

## Installation

With [npm][]:

```sh
npm i rose
```

With [Yarn][]:

```sh
yarn add rose
```

With [pnpm][]:

```sh
pnpm add rose
```

With [Bun][]:

```sh
bun add rose
```
